# Portfolio Website

Modern personal portfolio website for a data analyst and machine learning engineer, built with Next.js, React, Tailwind CSS, and Recharts.

## Run locally

```bash
npm install
npm run dev
```

## Structure

```text
app/
  ai-tools-lab/
  contact/
  projects/
    [slug]/
  resume/
  server-status/
  skills/
components/
  charts/
lib/
public/
```
